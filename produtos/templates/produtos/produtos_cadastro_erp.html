Consulta de cadastro

  • Dados vêm do mesmo cadastro usado no PDV (sincronizado com o ERP).
  • Esta tela não mostra saldo; alterações definitivas de produto continuam no ERP.
  • Busque por nome, marca ou código. Lista alfabética quando a busca está vazia.
  • Ao clicar na linha do produto (ou no ✏️), abre o modal de edição com os dados do espelho (a API de detalhe preenche custos, MVA, composição, etc.).
  • Com usuário logado, no detalhe do produto aparece o bloco Editar só no Agro (nome, preço, códigos, etc.); isso não altera o ERP.
  • Aba Grupos: cadastro local no Agro — um nome e um preço, com várias marcas/códigos de barras (preço igual para todas). Use Preencher do ERP na linha para copiar marca e código do espelho.
  • No modal de edição (✏️), a aba Validade e Lote grava data e lote no overlay Agro; entram no relatório de vencimentos.
  • F2 foca na busca (aba ERP) ou no nome do grupo; fora de campos, F1 volta ao PDV.
